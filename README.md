# TimeLineView

A simple Time Line View library for Android

# Screenshot

![TimeLineView Screenshot](Screenshot.gif)

# Setup

* In your `build.gradle` :

```gradle
repositories {
    maven { url "https://jitpack.io" }
}

dependencies {
    implementation 'com.github.nanjingboy:TimeLineView:1.2.0'
}
```
